### ‚ùì Why TurkHosts404?
**TurkHosts404** does not include global domain names, unlike other Turkish regional lists. It is designed as a **complementary list** that can be used alongside global lists. By focusing only on Turkish regional content, it offers a simple and effective list.

### üéØ What Do We List?
Entries included in our list are determined according to the following criteria:

| Category | Description |
| :--- | :--- |
| üö´ **Ads** | Web, app, and video-based advertising networks. |
| üïµÔ∏è **Trackers** | Tracking networks that monitor browsing behavior. |
| üìä **Analytics** | Data collection and usage statistics services. |
| üí• **Pop-ups** | Pop-up windows and full-screen ad warnings. |
| üç™ **Cookies** | Consent boxes and notifications that hinder browsing. |
